#####Supported Types


|Type       |Supported As | Shorthand |
|-----------|-------------|-----------|
|Object     |"Object"     |"Obj"      |
|Array      |"Array"      |"Arr"      |
|Function   |"Function"   |"Func"     |
|RegExp     |"RegExp"     |"Reg"      |
|Symbol     |"Symbol"     |"Str"      |
|null       |"null"       |           |
|undefined  |"undefined"  |           |
|string     |"string"     |"str"      |
|number     |"number"     |"num"      |
|integer    |"integer"    |"int"      |
|float      |"float"      |"flt"      |
|boolean    |"boolean"    |"bool"     |
